# XDG-SC-Platform
我们用kind模拟的k8s集群，并在其上模拟建立私有云以作为网络安全靶场。
如需在真实环境使用，建议使用真实的k8s集群作为基础平台。

所需组件

1. kind（模拟k8s集群）
2. kube-ovn（靶场的网络配置工具）
3. kubeVirt（k8s中使用的虚拟机启动工具）
4. go环境、docker、helm等工具，在安装前三个的教程中出现。
5. Python环境、Vue环境，本项目所需。
